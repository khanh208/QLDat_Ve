# Bao-cao-KTPM

Thu muc nay chua tai lieu phuc vu bai bao cao kiem thu phan mem.

## Tai lieu chinh

- `MUC TIEU KIEM THU.docx`: muc tieu va pham vi kiem thu
- `KE HOACH KIEM THU PHAN MEM-GK.docx`: test plan
- `CAC TRUONG HOP KIEM THU-GK.docx`: mo ta test case
- `TestCase-GK.xlsx`: bang test case chi tiet
- `TestReport-GK.xlsx`: tong hop ket qua test
- `Bao Cao Loi-GK.docx`: bug report
- `Ma tran lien ket yeu cau (Traceability Matrix).xlsx`: lien ket yeu cau va test case
- `PHAN-6-AUTOMATION-TESTING.md`: noi dung phan automation cho bao cao

## Thu muc automation

- `automation-artifacts`: log va bao cao tu cac lan chay automation

Mo `automation-artifacts/README.md` neu ban muon xem y nghia tung file trong do.


## Cap nhat moi

- `TestCase-GK.xlsx`: da bo sung `TC16-TC30`
- `TESTCASE-BO-SUNG-TC16-TC30.md`: tom tat noi dung cac testcase moi
